Input Control Elements (<input>)

Element <input> w HTML to kluczowy składnik formularzy służący do zbierania danych od użytkownika. Dzięki atrybutowi type może przyjmować różne formy - od zwykłego pola tekstowego po zaawansowane kontrolki jak wybór koloru, daty czy przesyłanie plików. Każdy typ inputa może mieć szereg atrybutów, które definiują jego zachowanie, walidację oraz wygląd.

  1. Podstawowe atrybuty
    • type - określa typ pola, np. text, password, email, file itp.
    • name - nazwa pola, przesyłana z danymi formularza.
    • id - unikalny identyfikator, często używany z label.
    • class - przypisuje klasę CSS.
    • value - domyślna wartość pola.
    • placeholder - podpowiedź, co użytkownik powinien wpisać.
    • required - pole obowiązkowe.
    • readonly - tylko do odczytu, nie można edytować.
    • disabled - pole wyłączone.
    • checked - domyślnie zaznaczone (dla radio/checkbox).
    • autocomplete - podpowiedzi przeglądarki (on/off).
    • multiple - pozwala na wiele wartości (dla file/email).
    • step, min, max - ograniczenia liczbowo-czasowe.
    • autofocus - automatyczne skupienie po załadowaniu strony.
    • pattern - walidacja poprzez wyrażenia regularne.
    • accept - określenie typów plików (dla file).
  2. Pole tekstowe (type="text")

    3. <label for="name">Imię:</label>
<input type="text" id="name" name="name" placeholder="Wpisz imię" required>

  3. Hasło (type="password")
    4. <label for="pass">Hasło:</label>
<input type="password" id="pass" name="password" required>
  4. Adres e-mail (type="email")
    5. <label for="email">E-mail:</label>
<input type="email" id="email" name="email" placeholder="twoj@email.com" required>
  5. Telefon (type="tel")
    6. <label for="phone">Telefon:</label>
<input type="tel" id="phone" name="phone" pattern="[0-9]{9}" placeholder="123456789">
  6. Data (type="date") 
    7. <label for="dob">Data urodzenia:</label>
<input type="date" id="dob" name="dob" min="1900-01-01" max="2025-12-31">
  7. Godzina (type="time")
    8. <label for="time">Godzina:</label>
<input type="time" id="time" name="meeting_time">
  8. Zakres liczbowy (type="range")
    9. <label for="volume">Głośność:</label>
<input type="range" id="volume" name="volume" min="0" max="100" step="5">

    13. <label for="website">Strona internetowa:</label>

  19. Tydzień i miesiąc (type="week" i type="month")
    20. <label for="week">Wybierz tydzień:</label>
<input type="week" id="week" name="week">
  20. Walidacja formularzy (HTML5)

    21. HTML5 pozwala na automatyczną walidację danych bez użycia JavaScript:

    • required - pole obowiązkowe.
    • pattern - dopasowanie do wyrażenia regularnego:
      <input type="text" pattern="[A-Za-z]{3,}" title="Wpisz co najmniej 3 litery.">
    • min, max, step - zakresy liczbowe:
      <input type="number" min="1" max="10" step="1">
    • minlength, maxlength - ograniczenie długości tekstu:
      <input type="text" minlength="3" maxlength="10">
    • Aby wyłączyć walidację dla przycisku:
      <input type="submit" formnovalidate>
