Class and ID

W HTML atrybuty class i id służą do identyfikowania elementów, co ułatwia ich stylowanie za pomocą CSS oraz manipulację przez JavaScript. Choć oba atrybuty mogą wyglądać podobnie, różnią się znaczeniem i sposobem użycia.

  1. Class - do grupowania

    2. Atrybut class może być przypisany do wielu elementów i nie musi być unikalny. Umożliwia grupowanie elementów, które mają mieć wspólne style. Aby przypisać klasę, wystarczy użyć atrybutu class w znaczniku HTML:

    <div class="example-class"></div>

    Jeśli chcemy przypisać więcej niż jedną klasę, wystarczy oddzielić je spacją: 

    <div class="class1 class2"></div>

    W CSS klasy selekcjonujemy za pomocą kropki: 

    .highlight {
  color: green;
}

    Ten styl zastosuje się do każdego elementu z klasą highlight, niezależnie od rodzaju znacznika (np. div, span). Jeśli chcemy zawęzić wybór tylko do konkretnych znaczników, możemy podać nazwę elementu i klasę: 

    div.highlight {
  color: green;
}
  2. Id - dla unikalnego oznaczenia

    3. Z kolei atrybut id służy do jednoznacznej identyfikacji konkretnego elementu na stronie. Każdy id powinien być unikalny w obrębie całego dokumentu. Przykład użycia:

    <div id="example-id"></div>

    Nie wolno przypisywać tego samego id różnym elementom:

    <div id="duplicate"></div>
<span id="duplicate"></span>  - Błąd!

    W CSS ID selekcjonujemy za pomocą znaku #:

    #example-id {
  color: green;
}

    W JavaScript elementy można pobierać przez id: 

    document.getElementById("example-id");

    Pamiętaj! Id musi być unikalny, nie może zawierać spacji. musi zaczynać się literą. Jeśli zduplikujemy Id to CSS nadal działa, ale jeśli spróbujemy w JS "var html = document.getElementById("aDiv").innerHTML;" to zwróci tylko pierwszy z nich.
