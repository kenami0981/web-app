Images

  1. Creating an image

    2. Aby stworzyć obrazek używamy znacznika <img> a następnie określamy lokalizację obrazka przy pomocny aytbuta src

    <img src="images/img.png" alt="Elmo w ogniu">

    Albo jeśli chcesz obrazek z adresu URL

    <img src="https://www.osvnews.com/wp-content/uploads/2024/02/20240213T0700-ELMO-CHALLENGE-OF-LENT-1772599-scaled.jpg" alt="Sesame Street muppet Elmo" width="250">
  2. Choosing alt text

    3. Tekst alternatywny alt jest czytany przez czytniki ekranu i widoczny, gdy obrazek się nie załaduje. Dobry alt powinien opisywać funkcję obrazka, a nie jego wygląd.

    Zły przykład
    <img src="anonymous.png" alt="Awatar anonimowego użytkownika">
    An anonymous user wrote: "lorem ipsum"
    Efekt:
    An anonymous user wrote: "lorem ipsum"
    Dobry przykład
    <img src="anonymous.png" alt="">
    An anonymous user wrote: "lorem ipsum"
    Efekt:
    An anonymous user wrote: "lorem ipsum"
    Ikony edycji / usuwania
    <a href="#"><img src="edit.png" alt="Edytuj"></a>
    <a href="#"><img src="delete.png" alt="Usuń"></a>
