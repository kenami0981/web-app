Sectioning Elements

Sectioning elements in HTML are used to define the structure of a document by grouping related content. This includes elements such as <nav>, <article>, <main>, <header>, <footer>, and <section>. Each of these elements helps in organizing a webpage for both developers and assistive technologies.

  1. Nav Element

    2. The <nav> element is used to group major navigational links. It can contain links to other sections of the page or external pages.

    Inline items:

    <nav>
    <a href="https://google.com">Google</a>
    <a href="https://www.yahoo.com">Yahoo!</a>
    <a href="https://www.bing.com">Bing</a>
</nav>
    Google Yahoo! Bing

    Use list items when needed:

    <nav role="navigation">
    <ul>
        <li><a href="https://google.com">Google</a></li>
        <li><a href="https://www.yahoo.com">Yahoo!</a></li>
        <li><a href="https://www.bing.com">Bing</a></li>
    </ul>
</nav>
    • Google
    • Yahoo!
    • Bing

    Avoid unnecessary usage of <nav> inside <footer>. The <footer> alone is sufficient:

    <footer>
    <a href="#">...</a>
</footer>
  2. Article Element

    3. The <article> element is used for self-contained, distributable content like blog posts or comments. It can be nested if related.

    <section>
    <article>
        <header>
            <h1>Blog Post</h1>
            <time datetime="2016-03-13">13th March 2016</time>
        </header>
        <p>...</p>
        <section>
            <h2>Comments</h2>
            <article id="user-comment-1">
                <p>Excellent!</p>
            </article>
        </section>
    </article>
</section>

    Avoid using <article> when content lacks context. Use <main> instead.

    <main>
    <p>I'm the main content, I don't need to belong to an article.</p>
</main>
  3. Main Element

    4. The <main> element holds the primary content of a webpage, should appear only once, and cannot be inside <header>, <footer>, or <nav>.

    <body>
    <header>
        <nav>...</nav>
    </header>
    <main>
        <h1>Individual Blog Post</h1>
        <article>
            <h2>References</h2>
            <p>...</p>
        </article>
    </main>
    <footer>...</footer>
</body>
  4. Header Element

    5. The <header> element provides introductory content and usually contains headings or navigation.

    <header>
    <p>Welcome to...</p>
    <h1>Voidwars!</h1>
</header>
        
    <article>
    <header>
        <h1>Flexbox: The definitive guide</h1>
    </header>
</article>
  5. Footer Element

    6. The <footer> element contains footer-related content like copyright info or links.

    <footer>
    <p>All rights reserved</p>
</footer>
  6. Section Element

    7. The <section> element groups thematically related content. Each section should have a heading.

    <article>
    <header>
        <h2>Blog Post</h2>
    </header>
    <p>Introduction...</p>
    <section>
        <h3>Chapter 1</h3>
        <p>...</p>
    </section>
    <section>
        <h3>Chapter 2</h3>
        <p>...</p>
    </section>
</article>

    Do not use <section> as a styling container. Use <div> for that purpose.
