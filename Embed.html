Znacznik <embed> w HTML5

  1. Opis znacznika

    Znacznik <embed> służy do osadzania treści zewnętrznych, zwykle multimedialnych lub interaktywnych, w dokumencie HTML. Najczęściej używany jest do osadzania plików wideo, dźwięków, animacji (np. Flash - obecnie przestarzałe), dokumentów PDF lub aplikacji napisanych w językach takich jak Java lub Silverlight (również przestarzałe).

    Element <embed> nie ma zamykającego tagu - jest tzw. void element.

  2. Podstawowe atrybuty

    1. src - Ścieżka (adres URL) do osadzanego zasobu.
    2. type - Typ MIME osadzanej zawartości (np. application/pdf, video/mp4).
    3. width - Szerokość osadzanej zawartości w pikselach.
    4. height - Wysokość osadzanej zawartości w pikselach.

  3. Podstawowe użycie

    Najprostsze użycie to osadzenie pliku bez określania typu MIME:

    <embed src="plik.swf">

    Obecnie jednak Flash nie jest wspierany, więc bardziej praktycznym przykładem będzie osadzenie wideo:

    <embed src="video.mp4" type="video/mp4" width="640" height="480">

  4. Uwaga o zgodności

    Niektóre przeglądarki mogą mieć ograniczone wsparcie dla <embed> w zależności od typu pliku lub konfiguracji zabezpieczeń. Alternatywnymi rozwiązaniami mogą być znaczniki <video>, <audio> lub <iframe>.

    Dlatego często stosuje się warstwę zapasową (fallback), np. za pomocą treści alternatywnej lub skryptów JavaScript.

  5. Przykłady praktyczne

    1. Osadzenie pliku PDF:
      <embed src="dokument.pdf" type="application/pdf" width="600" height="400">
    2. Osadzenie animacji SVG:
      <embed src="animacja.svg" type="image/svg+xml" width="300" height="300">
    3. Osadzenie dźwięku MP3 (choć <audio> jest preferowane):
      <embed src="muzyka.mp3" type="audio/mpeg" width="0" height="0">

  6. Różnice między <embed>, <object> i <iframe>

    1. <embed> - prostszy i bardziej bezpośredni, służy głównie do osadzania plików multimedialnych.
    2. <object> - bardziej rozbudowany, może zawierać zawartość alternatywną.
    3. <iframe> - osadza całą stronę internetową wewnątrz innej strony.
