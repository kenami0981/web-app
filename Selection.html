Notatka: Selection Menu Controls

  1. Select Menu

    Element <select> tworzy menu rozwijane.

    Przykład:

    <select name="">
    <option value="1">One</option>
    <option value="3">Three</option>
    <option value="2">Two</option>
    <option value="4">Four</option>
</select>

    Atrybut size określa ilość widocznych opcji. Wartość >1 zamienia rozwijane menu w okno z paskiem przewijania.

    multiple pozwala wybrać więcej niż jedną opcję. Domyślnie powoduje wyświetlenie listy w formie okna.

  2. Options

    Elementy <option> określają dostępne wybory. Przykład:

    <option value="option1">Some Option</option>

    Atrybuty:

    1. value - wartość przesyłana do serwera
    2. label - tekst widoczny w rozwijanym menu
    3. selected - domyślnie wybrana opcja

    Jeśli więcej niż jedna opcja ma selected, zostanie wybrana ostatnia z nich (w menu jednokrotnego wyboru). W przypadku multiple, wszystkie z selected będą zaznaczone.

  3. Option Groups

    <optgroup> pozwala grupować opcje w <select>. Przykład:

    <select name="">
    <option value="milk">Milk
    <optgroup label="Fruits">
        <option value="banana">Bananas
        <option value="strawberry">Strawberries
    </optgroup>
    <optgroup label="Vegetables" disabled>
        <option value="carrot">Carrots
        <option value="zucchini">Zucchini
    </optgroup>
</select>

    Grupa z atrybutem disabled uniemożliwia wybór wszystkich jej opcji.

  4. Datalist

    <datalist> służy do autouzupełniania danych w <input>. Przykład:

    <input list="Languages">
<datalist id="Languages">
    <option value="PHP">
    <option value="Perl">
    <option value="Python">
    <option value="Ruby">
    <option value="C+">
</datalist>

    Użytkownik widzi listę dostępnych propozycji podczas wpisywania danych.
