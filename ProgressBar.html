Progress Bar

  1. Progress

    2. Znacznik <progress> jest wykorzystywany aby pokwazać postęp wykonywanego zadania

    <progress value="22" max="100"></progress>
  2. Stylowanie progress bar

    3. Element <progress> pozwala wyświetlić pasek postępu. Można go stylować za pomocą selektora progress[value].

    Podstawowy styl paska postępu:

    progress[value] {
    width: 250px;
    height: 20px;
}

    Stylowanie w przeglądarkach:

    • Chrome, Safari, Opera

      Wymagane jest wyłączenie domyślnego stylu:

      progress[value] {
    -webkit-appearance: none;
    appearance: none;
}
progress[value]::-webkit-progress-bar {
    background-color: green;
}
    • Firefox

      Firefox stosuje dodatkowe style, np. ramkę - trzeba ją usunąć:

      progress[value] {
    -moz-appearance: none;
    appearance: none;
    border: none;
}
    • Internet Explorer 10+

      Nie obsługuje background-color. Użyj color:

      progress[value] {
    -webkit-appearance: none;
    -moz-appearance: none;
    appearance: none;
    border: none;
    width: 250px;
    height: 20px;
    color: blue;
}
  3. Fallback HTML - dla starszych przeglądarek:

    4. Jeśli przeglądarka nie obsługuje elementu <progress>, można użyć fallbacku wewnątrz niego:

    <progress max="100" value="20">
    <div class="progress-bar">
        <span style="width: 20%;">Progress: 20%</span>
    </div>
</progress>
    Progress: 20%

    Nowoczesne przeglądarki zignorują <div>, ale starsze (które nie rozpoznają <progress>) pokażą <div> jako alternatywę.
