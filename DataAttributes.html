Atrybuty danych (data attributes) w HTML5

Atrybuty danych (data-*) to funkcja wprowadzona w HTML5, która umożliwia przechowywanie niestandardowych informacji w elementach HTML. Są one przydatne szczególnie wtedy, gdy chcemy dodać do elementów dodatkowe dane, które mogą być wykorzystane np. przez JavaScript.

  1. Struktura atrybutu danych

    2. Każdy atrybut danych zaczyna się od prefiksu data-, a po nim następuje dowolna nazwa: 

    <div data-user-id="1234" data-role="admin"></div>

    Można w ten sposób zapisywać także dane w formacie JSON: 

    <div data-settings='{"theme":"dark"}'></div>
  2. Dostęp z poziomu JavaScript

    3. Nowoczesne przeglądarki pozwalają korzystać z właściwości dataset:

    const element = document.querySelector('div');
console.log(element.dataset.userId); // "1234"
element.dataset.role = "editor"; // zmiana wartości

    W przypadku starszych przeglądarek można używać metod getAttribute i setAttribute: 

    const id = element.getAttribute('data-user-id');
element.setAttribute('data-role', 'guest');
  3. Wsparcie przeglądarek

    4. Chociaż dataset nie działa w bardzo starych przeglądarkach, same atrybuty data-* są akceptowane. Przeglądarki ignorują nieznane atrybuty, ale ich nie usuwają, więc mogą być używane z poziomu JavaScript. 

    <img src="photo.jpg" value="test" />

    Tutaj value nie jest atrybutem dozwolonym w <img>, ale może być odczytany przez JS.

  4. Podsumowanie
    • Atrybuty data-* dodają niestandardowe dane do elementów HTML.
    • Dane są przechowywane jako łańcuchy znaków.
    • Dostęp przez dataset w nowych przeglądarkach, lub getAttribute/setAttribute w starszych.
    • Mogą zawierać np. JSON lub inne dane tekstowe.
    • Bezpieczne i zgodne ze standardami HTML5.
