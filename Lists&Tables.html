List and Tables

  1. Ordered List

    2. Numeryczną listę tworzysz używając znacznikó <ol> i <li> 

    <ol>
    <li>Banan</li>
    <li>Gruszka</li>
    <li>Czereśnia</li>
    <li>Truskawka</li>
</ol>

    1. Banan
    2. Gruszka
    3. Czereśnia
    4. Truskawka

    Jeśli natomiast chcesz zdefiniować od której liczby się zaczyna numerowanie to: 
    <ol start="10">
    <li>Banan</li>
    <li>Gruszka</li>
    <li>Czereśnia</li>
    <li>Truskawka</li>
</ol>

    1. Banan
    2. Gruszka
    3. Czereśnia
    4. Truskawka

    Odwrócona lista: 
    <ol reversed>
    <li>Banan</li>
    <li>Gruszka</li>
    <li>Czereśnia</li>
    <li>Truskawka</li>
</ol>

    1. Banan
    2. Gruszka
    3. Czereśnia
    4. Truskawka

  3. Neasted List (zagnieżdżona lista)

    4. <ul>
    <li>item 1</li>
    <li>item 2
        <ul>
            <li>sub-item 2.1</li>
            <li>sub-item 2.2</li>
        </ul>
    </li>
    <li>item 3</li>
</ul>

    • item 1
    • item 2
      • sub-item 2.1
      • sub-item 2.2
    • item 3

    • <caption> - Nadaje tytuł tabeli
    • <thead> - Sekcja nagłówka tabeli.
    • <tbody> - Główna zawartość (ciało) tabeli.
    • <tfoot> - Sekcja stopki - często z podsumowaniem.
    • colgroup> - Grupowanie kolumn do ewentualnej stylizacji.
    • colspan - Łączy komórki w poziomie (liczba kolumn do połączenia).
    • rowspan - Łączy komórki w pionie (liczba wierszy do połączenia).
    • scope="col" - Mówi, że nagłówek dotyczy kolumny.
    • scope="row" - Mówi, że nagłówek dotyczy wiersza.
